# 东莞鼎泰项目

## 第三方库

- loguru
  用于 日志记录
  `pip install loguru`

- numpy
  用于 数组计算
  `pip install numpy`

- scipy
  用于 数学计算，主要是旋转矩阵的计算
  `pip install scipy`

-tomlkit
  用于 解析 TOML 文件
  `pip install tomlkit`

- pyorbbecsdk2

奥比中光 SDK
用于 奥比中光 Gemini 305 相机的控制
`pip install pyorbbecsdk2`

- opencv-python
  用于 图像处理
  `pip install opencv-python`

-open3d
  用于 3D 模型处理
  `pip install open3d`
